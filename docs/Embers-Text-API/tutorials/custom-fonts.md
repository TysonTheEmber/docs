# 5) Custom fonts
1) Put your `.json` and `.ttf/.otf` in:
```
assets/emberstextapi/font/
```
2) Reference the font by its location ID in your NBT:
```
{font:"yourmod:my_font"}
```
3) Combine with color/gradient for unique looks.
